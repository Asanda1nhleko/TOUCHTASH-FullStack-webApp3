# TOUCHTASH - Salon Booking System

## Project Overview
TOUCHTASH is a full-stack web application that allows users to book salon appointments online and helps salon owners manage bookings, services, and customers.

---

## Features
- User registration and login
- Book appointments online
- Manage services and staff
- View booking history
- Admin dashboard

---

## Technologies Used
- HTML, CSS, JavaScript
- Bootstrap
- Node.js
- Express.js
- MySQL
- JWT Authentication

---

## Project Structure
```
Frontend/
Backend/
Database/
Documentation/
```

---

## How to Run the Project

### Backend
```bash
npm install
node server.js
```

### Frontend
Open the `index.html` file in the browser.

---

## Database
Import the SQL file from the `Database` folder into MySQL Workbench before running the project.
